## Projects

| Project | Description | Methods |
|---------|-------------|---------|
| [Sovereign Resource Dependency](https://leoss14.github.io/Portfolio/projects/capstone/page.html) | Country-level classification of natural resource exposure across 208 economies, built for Moody's Ratings to inform sovereign debt risk assessment | Clustering, decision trees, regularised regression; USGS + World Bank + UN Comtrade data |
| [US Monetary Policy Regimes](https://leoss14.github.io/Portfolio/projects/central-bank/page.html) | Regime-switching analysis of Federal Reserve policy rules across 70 years of macroeconomic data | Markov-switching models, Taylor rule estimation |
| [Chile Mineral Supply Chain](https://leoss14.github.io/Portfolio/projects/Chile/page.html) | Network mapping of 363 mining facilities and 17 critical commodities across Chile's export supply chain | Network analysis, Plotly visualisations, geospatial mapping |
| [Italian Export Complexity](https://leoss14.github.io/Portfolio/projects/export/page.html) | Analysis of Italian export diversification and economic complexity using international trade data | Economic complexity index, product space analysis |
| [Gender-Based Pricing (Pink Tax)](https://leoss14.github.io/Portfolio/projects/pink-tax/page.html) | ML classification pipeline identifying gender-based price differentials in UK supermarket products | XGBoost, LightGBM, regularised regression, validation diagnostics |
| [NYC Rideshare Demand](https://leoss14.github.io/Portfolio/projects/uber/page.html) | Spatial clustering of ride-hailing demand patterns in New York City, comparing 2018 and 2025 data | Spatial clustering, DBSCAN, Plotly mapping |
